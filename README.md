# 🗺️ Roam Pakistan

A lightweight, multi-page frontend web application built entirely using pure HTML. This project showcases the breathtaking beauty, historic landmarks, and top travel destinations of Pakistan using clean semantic structure.

---

## 🚀 Features

*   **Semantic Structure:** Built using pure HTML5 structural tags for a clean and logical layout.
*   **Multi-Page Navigation:** Seamless linking between pages (Home, Destinations, About Pakistan, Contact).
*   **Rich Media Showcase:** Integrated images, lists, and tables to present travel guides beautifully.
*   **Interactive Forms:** Includes a functional contact/feedback form layout for travelers.

---

## 🛠️ Tech Stack

*   **HTML5 Only:** 100% pure HTML structure without any external CSS frameworks or JavaScript libraries.

---

## 📂 Project Structure

Here is how the project files are organized:
*   `images` - the images folder
*   `index.html` - The main homepage introducing Pakistan tourism.
*   `destinations.html` - A detailed list of top tourist spots (Northern Areas, Heritage sites, etc.).
*   `about.html` - Cultural info, history, and travel tips.
*   `contact.html` - A simple contact form for user inquiries.
